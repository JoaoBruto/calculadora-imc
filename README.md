# 🧮 Calculadora de IMC Interativa

## 💡 Descrição
Este projeto é uma calculadora de IMC (Índice de Massa Corporal) desenvolvida em Python. O programa permite que o usuário:
- Insira seu peso e altura;
- Calcule seu IMC;
- Receba uma classificação com base no valor calculado.

Além disso, os resultados são registrados em um arquivo de texto para consulta posterior.

---

## 📂 Funcionalidades
- Calcula o IMC com base no peso e altura fornecidos pelo usuário.
- Classifica o IMC em categorias como:
  - **Magreza**
  - **Normal**
  - **Sobrepeso**
  - **Obesidade**
- Permite múltiplos cálculos em uma única execução.
- Registra todos os cálculos em um arquivo chamado `resultados_imc.txt`.

- **Praticidade**: Uma ferramenta simples para entender e acompanhar seu IMC.
